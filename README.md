# 🚀 John Kpoghomou | God Tier Flutter Portfolio

Welcome to my professional portfolio! This is a high-performance, responsive Single Page Application (SPA) built entirely with Flutter Web. It showcases my skills, projects, and professional journey as a Mobile and Web Developer.

---

## ✨ Features

- 📱 **Fully Responsive Design**: Optimized for Desktop, Tablet, and Mobile views.
- 🎨 **God Tier UI/UX**: Features glassmorphism, smooth animations (using `flutter_animate`), and interactive hover effects.
- ⚓ **Anchor Navigation**: Smooth scrolling between sections (About, Skills, Projects, Contact).
- 🌓 **Modern Aesthetics**: Clean typography and a professional blue/navy color palette.
- 📁 **Direct CV Access**: Integrated button to view or download my resume via Google Drive.
- 💬 **One-Tap Contact**: Direct links for WhatsApp, Email, and GitHub.

---

## 🛠️ Technologies Used

- **Framework**: [Flutter](https://flutter.dev) (Web)
- **Language**: [Dart](https://dart.dev)
- **Animation**: [Flutter Animate](https://pub.dev/packages/flutter_animate)
- **State Management**: Stateless/Stateful Widgets (Optimized for performance)
- **Backend/Services**: Firebase (Integration ready)
- **Deployment**: [GitHub Pages / Vercel]
