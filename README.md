---

## ✨ Points Forts•📱
Design Entièrement Responsive : Optimisé pour une expérience parfaite sur PC, Tablette et Mobile.•
🎨 Interface UI/UX Premium : Utilisation du glassmorphism,
d'animations fluides (via flutter_animate) et d'effets de survol interactifs.
•⚓ Navigation Fluide : Système d'ancres pour un défilement fluide entre les sections (Accueil, À propos, Compétences, Projets, Contact).
•🌓 Esthétique Moderne : Typographie épurée et palette de couleurs professionnelle (Bleu / Marine).
•📁 Accès Direct au CV : Bouton intégré pour consulter ou télécharger mon CV via Google Drive.
•💬 Contact Rapide : Liens directs vers WhatsApp, Email et GitHub.
